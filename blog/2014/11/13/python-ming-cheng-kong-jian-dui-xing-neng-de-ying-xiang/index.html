名称空间

Python 中变量的作用域有一个特别的名字叫做名称空间

名称空间呢, 有以下这些特点:

  • 每一个模块, 函数, 类, 实例, 都拥有一个独立的名称空间.
  • 每一个变量都会处在一个名称空间下.
  • 名称空间可以相互包含, 或者说有上下级关系
  • 在下级名称空间中可以访问上级名称空间的变量

以上名称空间, 从上向下都是包含关系.
